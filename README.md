# TRAE-Ollama-Bridge

Wrap local Ollama as an OpenAI-compatible API to bypass fixed-model restrictions in IDEs like TRAE. For full documentation, see:

- 中文文档: `docs/README.zh-CN.md`
- English Docs: `docs/README.en.md`

Quick start: double-click `Start-Bridge.bat`, open the Web UI, and manage model mappings or interception policy as needed.
